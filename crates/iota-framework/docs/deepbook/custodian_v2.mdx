## Resource `AccountCap`

Capability granting permission to access an entry in <code>
<Link href="custodian_v2.md#0xdee9_custodian_v2_Custodian">Custodian</Link>.account_balances</code>.
Calling <code>
mint_account_cap</code> creates an "admin account cap" such that id == owner with
the permission to both access funds and create new <code>
<Link href="custodian_v2.md#0xdee9_custodian_v2_AccountCap">AccountCap</Link></code>s.
Calling <code>
create_child_account_cap</code> creates a "child account cap" such that id != owner
that can access funds, but cannot create new <code>
<Link href="custodian_v2.md#0xdee9_custodian_v2_AccountCap">AccountCap</Link></code>s.

<b>struct</b> <Link href="custodian_v2.md#0xdee9_custodian_v2_AccountCap">AccountCap</Link> <b>has</b> store, key
