## Function `extract_assets`

The function extracts assets from a legacy <code>
<a href="alias_output.md#0x107a_alias_output_AliasOutput">AliasOutput</a></code>.
- returns the coin Balance,
- the native tokens Bag,
- and the <code>
Alias</code> object that persists the AliasID=ObjectID from Stardust.
