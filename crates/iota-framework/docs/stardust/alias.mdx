## Resource `Alias`

The persisted Alias object from Stardust, without tokens and assets.
Outputs owned the AliasID/Address in Stardust will be sent to this object and
have to be received via this object once extracted from <code>
AliasOutput</code>.


<pre><code>
<b>struct</b> <Link href="alias#0x107a_alias_Alias">Alias</Link> <b>has</b> store, key
</code></pre>

 This is the AliasID from Stardust.
