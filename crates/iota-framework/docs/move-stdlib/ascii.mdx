## Struct `String`

The 
<Link to="../move-stdlib/ascii#0x1_ascii_String">String</Link> struct holds a vector of bytes that all represent
valid ASCII characters. Note that these ASCII characters may not all
be printable. To determine if a 
<Link to="../move-stdlib/ascii#0x1_ascii_String">String</Link> contains only "printable"
characters you should use the 
all_characters_printable predicate
defined in this module.
