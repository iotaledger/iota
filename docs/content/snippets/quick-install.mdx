Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux) or [Chocolatey](https://chocolatey.org/) (Windows) to install Iota.

<Tabs groupId="operating-systems">

<TabItem label="Homebrew" value="mac">

```bash
brew install iota
```

</TabItem>

<TabItem label="Chocolatey" value="win">

```shell
choco install iota
```

Find more [versions of Iota for Windows](https://community.chocolatey.org/packages/iota) on the Chocolatey community website.

</TabItem>

</Tabs>
