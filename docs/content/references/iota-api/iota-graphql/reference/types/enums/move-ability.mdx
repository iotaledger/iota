Abilities are keywords in Iota Move that define how types behave at the
compiler level.


```graphql
enum MoveAbility {
  COPY
  DROP
  KEY
  STORE
}
```




### Values

#### [<code style={{ fontWeight: 'normal' }}>MoveAbility.<b>COPY</b></code>](#)  
> 
> 
> Enables values to be copied.
> 

#### [<code style={{ fontWeight: 'normal' }}>MoveAbility.<b>DROP</b></code>](#)  
> 
> 
> Enables values to be popped/dropped.
> 

#### [<code style={{ fontWeight: 'normal' }}>MoveAbility.<b>KEY</b></code>](#)  
> 
> 
> Enables values to be held directly in global storage.
> 

#### [<code style={{ fontWeight: 'normal' }}>MoveAbility.<b>STORE</b></code>](#)  
> 
> 
> Enables values to be held inside a struct in global storage.
>
