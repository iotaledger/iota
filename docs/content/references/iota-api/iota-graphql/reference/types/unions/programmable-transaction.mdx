A single transaction, or command, in the programmable transaction block.


```graphql
union ProgrammableTransaction = MoveCallTransaction | TransferObjectsTransaction | SplitCoinsTransaction | MergeCoinsTransaction | PublishTransaction | UpgradeTransaction | MakeMoveVecTransaction
```




### Possible types

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>MoveCallTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/move-call-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> A call to either an entry or a public Move function.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>TransferObjectsTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/transfer-objects-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Transfers `inputs` to `address`. All inputs must have the `store` ability (allows public
> transfer) and must not be previously immutable or shared.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>SplitCoinsTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/split-coins-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Splits off coins with denominations in `amounts` from `coin`, returning multiple results (as
> many as there are amounts.)
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>MergeCoinsTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/merge-coins-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Merges `coins` into the first `coin` (produces no results).
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>PublishTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/publish-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Publishes a Move Package.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>UpgradeTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/upgrade-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Upgrades a Move Package.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProgrammableTransaction.<b>MakeMoveVecTransaction</b></code>](/references/iota-api/iota-graphql/reference/types/objects/make-move-vec-transaction) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Create a vector (possibly empty).
>
