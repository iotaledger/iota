---
title: Iota and Community SDKs
description: Collection of SDKs and utilities for developing on Iota using various programming languages.
---

Iota provides developer kits that act as wrappers for the Iota API. The Iota community broadens the code coverage with its own set of developer kits targeting the Iota blockchain. 

## Iota SDKs

<Cards>
<Card title="dApp Kit" href="https://docs.iota.org/references/ts-sdk/dapp-kit">
    A web frontend SDK that interacts with the Iota API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the Iota API with Rust, using the `iota-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://github.com/iotaledger/iota/iota_sdk/index.html">
    Auto-generated documentation for the `iota-sdk` crate in the Iota repository.
</Card>
<Card title="TypeScript SDK" href="https://docs.iota.org/references/ts-sdk/typescript">
    TypeScript SDK for integrating Iota in your TS dApps.
</Card>
<Card title="zkSend SDK" href="https://docs.iota.org/references/ts-sdk/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/IotaFansCN/suiue">
    Iota dApp Kit for the Vue framework.
</Card>
<Card title="Dart SDK" href="https://github.com/mofalabs/iota">
    A cross-platform Iota SDK for mobile, web, and desktop.
</Card>
<Card title="Go SDK" href="https://github.com/block-vision/iota-go-sdk">
    SDK for developing for Iota using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/ksui">
    Ksui is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a Iota Full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pysui">
    pysui is a Python client for developing on the Iota blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/suikit">
    IotaKit is a Swift SDK natively designed for developing on the Iota blockchain.
</Card>
</Cards>
